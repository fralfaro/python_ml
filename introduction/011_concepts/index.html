    Motivaci贸n露

    La raz贸n de utilizar dos conjuntos de datos separados para entrenar y evaluar el modelo es evitar el sobreajuste (overfitting) del modelo. El sobreajuste ocurre cuando un modelo se ajusta demasiado bien a los datos de entrenamiento y no generaliza bien a nuevos datos. Si se utilizan los mismos datos para entrenar y evaluar el modelo, es probable que el modelo tenga un buen rendimiento en los datos de entrenamiento pero un rendimiento pobre en datos nuevos.

    Reglas de separaci贸n露

    En general, se divide el conjunto de datos de entrada en dos conjuntos: el conjunto de entrenamiento y el conjunto de prueba. El conjunto de entrenamiento suele ser el 70-80% del conjunto de datos y el conjunto de prueba el 20-30%. La proporci贸n puede variar seg煤n el tama帽o del conjunto de datos y la complejidad del modelo.

    A modo de regla emp铆rica, se considerar谩 el tama帽o 贸ptimo basado en la siguiente tabla:

    entre 1.000- 100.000 80% 20%
    mayor a 100.000 99% 1%

    M谩s informaci贸n露

    Es importante destacar que, en algunos casos, se utiliza un tercer conjunto de datos, llamado conjunto de validaci贸n (validation set), que se utiliza para ajustar los hiperpar谩metros del modelo y evitar el sobreajuste. En este caso,

    • el conjunto de entrenamiento se utiliza para ajustar los par谩metros del modelo,
    • el conjunto de validaci贸n se utiliza para ajustar los hiperpar谩metros
    • y el conjunto de prueba se utiliza para evaluar el rendimiento final del modelo.

    M茅tricas de estimaci贸n

    En este contexto, se busca maximizar el n煤mero al m谩ximo la suma de los elementos TP Y TN, mientras que se busca disminuir la suma de los elementos de FN y FP. Para esto se definen las siguientes m茅tricas:

    1. Accuracy

    $$accuracy(y,\hat{y}) = \dfrac{TP+TN}{TP+TN+FP+FN}$$

    1. Recall:

    $$recall(y,\hat{y}) = \dfrac{TP}{TP+FN}$$

    1. Precision:

    $$precision(y,\hat{y}) = \dfrac{TP}{TP+FP} $$

    1. F-score:

    $$fscore(y,\hat{y}) = 2\times \dfrac{precision(y,\hat{y})\times recall(y,\hat{y})}{precision(y,\hat{y})+recall(y,\hat{y})} $$
